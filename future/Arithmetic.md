  + 栈


    + **只允许在一端进行插入或删除** 操作的 **线性表**

    + **卡特兰数**：$n个不同元素进栈，出栈元素不同排列的个数为\frac{1}{n+1} C^n_{2n}$

    + 基本操作

      ```c
      // 初始化一个空栈
      InitStack(&S)
        
      // 判断一个栈是否为空，若栈S为空则返回true，否则返回false
      StackEmpty(S)
        
      // 进栈，若栈S未满，则将x加入使之成为新栈顶
      Push(&S, x)
        
      // 出栈，若栈S非空，则弹出栈顶元素，并用x返回
      Pop(&S, &x)
        
      // 读栈顶元素，若栈S非空，则用x返回栈顶元素
      GetTop(S, &x)
        
      // 销毁栈，并释放栈S占用的存储空间
      DestroyStack(&S)
      ```

      > 解答算法题时，若题干未做出限制，则可直接使用这些基本的操作函数
